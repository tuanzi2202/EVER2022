# 数据类型
## 整型数据
short     占2字节

int       占4字节

long      32位占4字节/64位占8字节

long long 占8字节

size of(数据或数据类型) 求当前类型占几个字节

unsigned/signed 无符号/有符号，修饰整型数据

int <=> signed int 有符号整型，最高bit位为符号位，剩下为数据位

unsigned int  无符号整型，全部bit位都为数据位

## 字符型数据
char  占4字节

常用ascii码：'0':48 , 'A':65 , 'a':97

## 浮点型数据
float 占4字节

double  占8字节

long double 占16字节
